# On-Demand-Exam-System
This is a Python,Django Project of on demand exam system in which  students can select date and subject for giving their exam

How to run this code

First you have to install latest version python

then open command prompt and install django by writing

-->   pip install django

then migrate databases

-->  python manage.py makemigrations
-->  python manage.py migrate

then run the server

-->  python manage.py runserver

then copy the localhost url and go to browser
